Red Social para Fotógrafos

← Volver al portafolio

Resumen

Aplicación enfocada en fotógrafos para compartir trabajos, recibir feedback y participar en retos fotográficos.

Propósito

Crear una comunidad especializada donde fotógrafos puedan exponer su trabajo y mejorar mediante interacción social.

Explicación técnica

  • Lenguaje: Java para compatibilidad con dispositivos más antiguos.
  • Red: Retrofit para consumo de APIs RESTful.
  • Imágenes: Glide para carga y cache de imágenes optimizadas.
  • Backend: API REST en Node.js + MongoDB (ejemplo).
  • Autenticación: OAuth2 / JWT.
