App de Gestión de Tareas

← Volver al portafolio

Resumen

Aplicación Android para gestionar tareas, con sincronización en la nube, notificaciones y soporte offline.

Propósito

Facilitar la organización personal y profesional con recordatorios inteligentes y sincronización entre dispositivos.

Explicación técnica

  • Arquitectura: MVVM con capas limpias.
  • Lenguaje: Kotlin.
  • Persistencia local: Room para base de datos local y caching.
  • Sincronización: Firebase Realtime Database / Firestore para sincronización y autenticación.
  • Notificaciones: WorkManager + Notification Channels para recordatorios programados.
  • Pruebas: Unit tests con JUnit y pruebas de UI con Espresso.
