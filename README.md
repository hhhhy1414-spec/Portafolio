# Portafolio de Ingeniero de Software Android

Este es un sitio web de portafolio profesional para un ingeniero de software especializado en el desarrollo de aplicaciones Android.

## Características

- **Diseño Profesional**: Interfaz limpia y moderna con diseño responsive.
- **Secciones**:
  - Sobre Mí: Breve introducción y enlaces a GitHub y LinkedIn.
  - Proyectos: Tarjetas con portada, descripción y tecnologías utilizadas.
  - Contacto: Información de contacto y enlaces sociales.
- **Tecnologías Utilizadas**:
  - HTML5
  - CSS3
  - JavaScript (ES6+)

## Cómo Usar

1. Abre el archivo `index.html` en tu navegador web preferido.
2. Navega por las diferentes secciones usando el menú de navegación.
3. Los proyectos se muestran como tarjetas interactivas.

## Personalización

Para personalizar este portafolio:

1. Edita `index.html` para cambiar el contenido (nombre, descripción, proyectos, etc.).
2. Modifica `styles.css` para ajustar colores, fuentes y estilos.
3. Actualiza `script.js` si necesitas agregar más funcionalidades JavaScript.
4. Reemplaza las imágenes placeholder con imágenes reales de tus proyectos.
