App de Fitness Personalizado

← Volver al portafolio

Resumen

Aplicación que genera rutinas de ejercicio, adapta planes según progreso y permite seguimiento de métricas.

Propósito

Ayudar a usuarios a conseguir objetivos de salud y forma física mediante planes personalizados y seguimiento automático.

Explicación técnica

  • Lenguaje: Kotlin con Jetpack Compose para UI moderna.
  • Estado: ViewModel + LiveData / StateFlow.
  • Persistencia: SQLite/Room para datos locales y sync opcional en la nube.
  • Integración: Sensores y Google Fit API para métricas de salud.
